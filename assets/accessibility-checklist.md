# Accessibility Checklist

Run this before lock. Accessibility is design quality, not a checkbox.

---

## Captioning

### Live in-room
- ☐ Captioning visible to audience (open caption screen or AR)?
- ☐ Single source of truth (one captioner / one ASR)?
- ☐ Glossary pre-loaded for product names, exec names, technical terms?
- ☐ Human captioner present for high-stakes segments?

### Live stream
- ☐ ASR captions enabled?
- ☐ Human captioner backup?
- ☐ Multiple languages? Which?
- ☐ Caption styling readable (size, contrast, position)?

### On-demand
- ☐ Edited SRT delivered with the master cut?
- ☐ Corrections applied from live errors?
- ☐ All cutdowns include burned-in captions?
- ☐ All cutdowns include separate SRT for platforms that prefer them?

## Audio description
- ☐ Is an audio description track needed (any silent visual moments carrying meaning)?
- ☐ Separate audio channel for AD on stream?
- ☐ AD script reviewed for tone with broader content?

## Sign language interpretation
- ☐ ASL / BSL / local SL interpreter booked?
- ☐ Interpreter visible on stage AND in stream picture-in-picture?
- ☐ Interpreter has the script in advance?
- ☐ Interpreter has the glossary?

## Visual design
- ☐ Body text contrast ≥ 4.5:1 (WCAG AA)?
- ☐ Headline contrast ≥ 3:1?
- ☐ No color-only encoding in charts (use shape / pattern / label too)?
- ☐ Font size readable from back row (test at venue or simulate)?
- ☐ Motion / flashing content within safe limits (no rapid flashing > 3Hz)?

## Translation / localization
- ☐ Which languages are we serving?
- ☐ Live translation (interpreter booths) or AI?
- ☐ Cutdown subtitles in target languages?
- ☐ Cultural review of metaphors, idioms, humor?
- ☐ Numbers / dates / currency formats localized?

## Inclusive language
- ☐ Script reviewed for ableist language?
- ☐ Gender-neutral phrasing where appropriate?
- ☐ Customer / employee imagery represents audience diversity?
- ☐ Pronouns confirmed for every named speaker?
- ☐ Names pronounced correctly (phonetic guides in prompter)?

## Physical accessibility (venue / stage)
- ☐ Reserved seating for wheelchair users, low-vision, hard-of-hearing?
- ☐ Sign language interpreter sightlines confirmed?
- ☐ Quiet room / sensory-friendly space available?
- ☐ Stage ramp / accessible path for any guest who needs it?

## Communications
- ☐ Pre-event communications mention accessibility offerings?
- ☐ Contact for accessibility requests included in event comms?
- ☐ Day-of accessibility lead identified and reachable?
